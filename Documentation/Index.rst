Search Word Suggestion / Autocompletion
=======================================

This Plugin extends the TYPO3 CMS with an easy and simple Map Extension
use for company roadmap on websites also for responsive pages, based on the google maps API

Features
--------
* Positioning of Map Marker with drag & drop and via adress field
* Custom Map Marker (Pin) for every Adress
* Anchor Position adjusting if the Marker Image is not centered
* Customizing Frontend and Design, responsive Map are not Problem
